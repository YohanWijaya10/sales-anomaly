# Sales Dashboard

Dashboard untuk memantau kunjungan sales, performa penjualan, dan insight harian.

## Mulai Cepat

Jalankan server development:

```bash
npm run dev
```

Buka `http://localhost:3000` di browser.

## Catatan

- Halaman utama ada di `app/page.tsx`.
- Halaman dashboard ada di `app/dashboard/page.tsx`.

## Deploy

Kamu bisa deploy dengan platform pilihanmu (misalnya Vercel). Pastikan environment variables sudah diatur.
